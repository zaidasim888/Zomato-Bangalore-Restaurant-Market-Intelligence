# Restaurant Market Entry Strategy Analysis – Bangalore

A Power BI business intelligence project analyzing 12,111 Bangalore restaurants to identify the optimal location, cuisine, pricing strategy, and competitive positioning for a new restaurant venture.

## Objectives
- Identify high-potential restaurant locations
- Analyze cuisine demand and competition
- Evaluate pricing trends across market segments
- Assess competitive advantages through online ordering and table booking

## Tools Used
- Power BI
- Power Query
- DAX

## Key Insights
- Bellandur and Sarjapur Road emerged as promising market-entry locations.
- Biryani showed strong demand with manageable competition.
- Recommended pricing range: ₹450–₹650 for two.
- Only 7.8% of restaurants offer table booking, presenting a competitive opportunity.
